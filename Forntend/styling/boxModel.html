Box Model

box model is a layout model for html elements

which describe the content , padding , broder and margin of the element

Content

the content is a area where the text and images and videos appear

padding

the padding area is the space between the content and the broder

to change the padding , use the padding property in the css

 #box1{
          padding:20px; 
          padding: 10px 20px;
          padding:10px 20px 5px 15px;
          padding-top:10px;
          padding-bottom:5px;
          padding-left:15px;
          padding-right:10px;
    }

broder

the broder area is the line/layout around the content

to change the broder , use the broder property inthe css

            #box1{
                broder:1px solid black;
                border-width:5px;
                border
            }

margin

the margin area is the space outside the broder

to change the margin,use the margin property in the css 

 
            #box1{
                margin:30px;
            }

Note

            
             box- box-sizing property is used to include the padding and broder in the total width and height 
             the default value is content-box
             #box1{
                box-sizing: border-box;

             }
             
    
                
  •  content box- width and height inculdes content only
    • 
                
  •  border box- width and height inculdes padding and broder
    • 
             

             #box1{
                width:200px;
                height:100px;
                padding:20px;
                border:4px;
               box-sizing: content-box; /* total width = 200 +20+4+4-248px, total height = 100+20+20+4+4-148px */
               box-sizing: border-box;/total width = 200[px, total height = 100px */
               
             }
